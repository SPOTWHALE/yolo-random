# Candidates

Curated list of public repos that **welcome outside contributions** and run a
`good first issue` / `help wanted` program. The `/yolo random` skill only ever
targets repos on this list.

Each repo declares a **`Method`** — how it wants fixes submitted:

- `comment` — issue comment with a diff only (no fork, no PR). Non-invasive.
- `pr` — a real pull request (fork + PR).
- `both` — either is fine; the user's preference decides.

## How to qualify a repo before adding it

- Has an open `good first issue` or `help wanted` label with real issues.
- `CONTRIBUTING.md` does **not** ban AI/automated contributions, and (for `pr`)
  needs no unsigned CLA.
- Actively maintained (a commit within the last ~6 months).
- You (the curator) set `Method` to match what the maintainers actually want.
